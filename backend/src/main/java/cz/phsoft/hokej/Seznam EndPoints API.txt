MATCH
@RequestMapping("/api/matches")
GET
/api/matches                    - všechny zápasy        - Admin, Manager
/api/matches/matchDetail/{id}   - detail zápasu         - isAuthenticated
/api/matches/next               - první aktuální zápas  - isAuthenticated
/api/matches/admin/upcoming     - nadcházející zápasy   - Admin, Manager
/api/matches/me/upcoming        - nadcházející zápasy   - isAuthenticated
/api/matches/me/upcoming-overview - nadcházející zápasy hráče - přehled - isAuthenticated
/api/matches/me/all-passed      - uplynulé zápasy hráče - přehled - isAuthenticated
/api/matches/admin/past         - uplynulé zápasy       - Admin, Manager
/api/matches/admin/{id}        - zápas dle id zápasu   - Admin, Manager
/api/matches/admin/available-for-player/{player id} - zápasy kde hráč není Neaktivní - Admin, Manager

POST
@PostMapping - /api/matches/         - přidat zápas      - Admin, Manager
@PutMapping  - /api/matches/admin/{id}    - editovat zápas    - Admin, Manager
@DeleteMapping - /api/matches/admin/{id}  - smazat zápas      - Admin
