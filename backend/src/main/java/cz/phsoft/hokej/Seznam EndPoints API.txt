----------------------
MATCH
@RequestMapping("/api/matches")
GET
/api/matches                    - všechny zápasy        - Admin, Manager
/api/matches/matchDetail/{id}   - detail zápasu
/api/matches/next               - první aktuální zápas
/api/matches/upcoming           - nadcházející zápasy   - Admin, Manager
/api/matches/past               - uplynulé zápasy       - Admin, Manager
/{id}                           - zápas dle id zápasu
/api/matches/available-for-player/{id} - zápasy kde hráč není Neaktivní - id_player
/api/matches/player/{playerId}/upcoming - zápasy pro aktivního hráče dle id s filtrování upcoming dle typuHráče (vše, 2, 1)

POST
@PostMapping - /api/matches         - přidat zápas      - Admin, Manager
@PutMapping  - /api/matches/{id}    - editovat zápas    - Admin, Manager
@DeleteMapping - /api/matches/{id}  - smazat zápas      - Admin, Manager

REGISTRACE - MatchRegistration
@RequestMapping("/api/registrations")

GET
/api/registrations/all                      - všechny registrace                - Admin, Manager
/api/registrations/for-player/{playerId}    - všechny registrace dle id hráče
/api/registrations/for-match/{matchId}      - všechny registrace dle id zápasu  - Admin, Manager
/api/registrations//no-response/{matchId}   - seznam hráčů, kteří nereagovali   - Admin, Manager

POST
@PostMapping - /api/registrations/upsert      - vytvoří registrace dle id zápasu a id hráče
@PostMapping - /api/registrations/upsert    - vytvoří záznam o zrušení registrace (odregistrování)
                                                 dle id zápasu a id hráče
@PostMapping - /api/registrations/upsert        - vytvoří záznam o omluvení se ze zápasu

AKTIVNÍ / NEAKTIVNÍ HRÁČ - PlayerInactivity

GET
